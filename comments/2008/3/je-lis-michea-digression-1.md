Paul .ca @ 2008-03-10 21:19:36

> Par progrès, on entend souvent évolution positive. Mais qu’est-ce qui est positif ? Il est impossible de le définir dans l’absolu.

Oui Thierry donc c’est relatif à quelque chose comme l’humanité ou un groupe de personnes. Ce qui positif à un instant T1 pour les uns peut être négatif pour d’autres à T1 ou T2.

Ensuite mieux vaut être dans les groupes qui bénéficient de ces progrès (économique, technique, médicale, environnement ... ) plutôt que de philosopher sur le progrès.

J’intègre le facteur temps car souvent avec une découverte scientifique on parle de "progrès pour l’humanité" sans connaitre tjs bien les impacts négatifs qui peuvent avec le temps en s’accumulant relativiser lourdement le "positif" initial pour le groupe ou la région bénéficiaire.
