# Guide de lecture

Voici quatre solutions pour découvrir dans les meilleures conditions mon feuilleton [*Résistants*](http://tcrouzet.com/resistants/) (le roman pour tout comprendre des monstres microscopiques qui nous menacent).<span id="more-43797"></span>

### 1/ Vous aimez le confort

1. Inscrivez-vous sur [Wattpad](https://www.wattpad.com).

2. [Cliquez sur ce lien](https://www.wattpad.com/story/61038614-r%C3%A9sistants) qui vous amènera sur la fiche Wattpad de [*Résistants*](http://w.tt/22GUUWY).

3. Ajoutez [*Résistants*](http://w.tt/22GUUWY) à votre liste de lecture (le bouton +).

4. Chaque fois que vous reviendrez sur Wattpad, vous retrouverez le feuilleton où vous l’avez laissé.

5. Pour une lecture plus confortable, installez l’application Wattpad sur votre téléphone ou tablette. Vous pourrez interagir très facilement avec moi et les autres lecteurs. C’est sans doute aujourd’hui la meilleure façon de lire en ligne.

### 2/ Vous n'aimez pas vous inscrire

![Sur le blog](http://tcrouzet.comhttps://tcrouzet.com/images_tc/2016/04/r3-600x196.jpg)

[Lisez le roman sur mon blog.](http://tcrouzet.com/2016/04/18/episode-1-paradise/) Il se déroule comme un rouleau et les chapitres se succèdent dans l'ordre chronologique. Pour reprendre la lecture, cliquez sur le bouton rouge "Résistants 1/n", il affiche un sommaire. Vous pouvez laisser des commentaires en cliquant sur le "+" associé à chaque chapitre.
