# On fignole la POU100

[Après avoir bouclé les recos sur les parcours 100 km et 160 miles](https://tcrouzet.com/2026/03/05/POU100-2026/), nous avons enfin concocté une variante roulante pour boucler le 250 km ([quasi complètes, restent 5 places](https://www.helloasso.com/associations/ec-poussan/evenements/pou100-100-miles-100-km-ou-250km-gravel-de-poussan-2026)). C’est parfois très compliqué de trouver les itinéraires hors asphalte pour les gravels. Nous adorons la pratique, mais elle est souvent restrictive. Alors nous sommes doublement heureux quand nous dénichons une solution quasi impossible à l’équation. Voici quelques photos pour le plaisir de partager. Rien de plat dans ce secteur. [Nous avons grimpé près de 2000 m pour 90 km.](https://www.strava.com/activities/17698013934)
